GNPX

Sudoku analysis application(win)
  • GNPX solves the Sudoku problem using analysis algorithm of this site.
  • Application of analysis algorithm can be individually turned on/off, and analysis order can be changed. Sequential step analysis and full automatic analysis are possible.
  • For multiple solution analysis, when multiple analysis algorithms are established, all of them (with upper limit) are obtained.
  • For problem generation, you specify a pattern and create a problem. Depending on the pattern, search time may be longer.
  • In order to input problems manually, select "Create/Manual" and click the target cell of the board. It can be set with utmost effort.
  • There is no manual. You can operate within ordinary imagination.
  • Win10 and over is recommended as the execution environment.
  • V3.06, web camera input is installed. OpenCvSharpExtern.dll is required in [execution folder]\dll\x64 . Please download and install.
  • V3.07 has no camera input function.  The camera function will be another system.

GNPX proj

Sudoku analysis application Project
  • GNPX win application's C # source and project.
  • It can be used as a platform to learn and develop Sudoku analysis algorithm in C#.
    If you build your own algorithm part, you can do full automatic analysis, step analysis, multiple solution analysis, problem creation.
  • The development environment is Win10 or higher, VS 2015/2017, C#6.0.
  • Win10 and over is recommended as the execution environment.

  • Place the compiled executable file in the Sudoku_App folder.
